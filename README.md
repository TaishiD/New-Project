# DeGoat Jewelry

This is the static landing page for **DeGoat Jewelry**.

- Built with React + Tailwind (via CDN)
- Embedded images are inline (base64), so no extra assets needed
- Features pricing tiers, reviews, contact info, and an offer request modal

## Deployment

This site is ready for GitHub Pages.

1. Upload `index.html` (and this README if you like) into your repo.
2. In your repository settings → Pages → choose "Deploy from branch".
3. Select branch: `main`, folder: `/ (root)` → Save.
4. Your site will be live in about a minute.

For a user site, name the repo **TaishiD.github.io**.  
For a project site, name it whatever you like (e.g., `degoat-jewelry`).
